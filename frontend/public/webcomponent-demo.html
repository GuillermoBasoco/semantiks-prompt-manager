Semantiks Web Component Demo

This page demonstrates the custom element without an iframe. It uses your app's /api/ask and the currently active prompt by default.

Floating widget

Appears at bottom-right. No extra markup needed beyond the tag itself.

Inline widget

Renders in-place (non-floating). You can place it anywhere in your layout.

Target a specific prompt (optional)

Replace 123 with a real prompt id to override the active prompt.
